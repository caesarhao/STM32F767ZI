## STM32F767ZI Chip feature
* High performance
* Arm Cortex-M7
* CPU 216 MHz
* L1 cache
* LQFP144 package
* 32.768 kHz crystal oscillator
* Flash 2 MB
* SDRAM 512 kB


## Environment
### Program and debug
Under Fedora, install the packages **stlink** and **stlink-gui**
### Compilation
Under Fedora, install the package **arm-none-eabi-gcc**
